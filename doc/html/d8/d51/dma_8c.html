void DMA_Config ( )

This function configure DMA for transferring data from peripheral to in memory buffer (DMA_DIR_PeripheralToMemory). Buffered stream is used in direct mode (DMA_FIFOMode_Disable) so after every data transfer from peripheral to FIFO, corresponding data is immediately stored in destination. To avoid saturating the FIFO, the corresponding stream is configured with a high priority (DMA_Priority_High). Memory and peripheral bursts are specified to be single burst, so in a single non interruptible transaction only one single data is transferred. Circular mode is selected (DMA_Mode_Circular) to handle circular buffer and continuous data flows.

This function is used to initialize DMA.
